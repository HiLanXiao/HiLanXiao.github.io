shell编程

一、shell 编程的基本过程

  • shell 编程的基本过程分为三步:
    • 建立 shell 文件
      • 包含任意多行操作系统命令或shell命令的文本文件
    • 赋予shell文件执行权限
      • 用chmod命令修改权限
    • 执行shell文件
      • 直接在命令行上调用shell程序
    • 实例
      • 建立shell文件 (可用任何建立文本文件的方法):
        • $ cat prog1
        • who | grep $1
      • 赋予执行权限: (初始文本文件无执行权限)
        • $ chmod 740 prog1
      • 执行该shell程序
        • $ prog1 student5
        • prog1: not found
        • (shell在标准搜索目录中找不到prog1命令)
      • 指定路径或修改环境变量PATH后执行shell程序
        • $ ./prog1 student5
        • student5 tty06 Feb 8 09:12

二、 shell语句包括三大类

  • 说明性语句
    • 以#号开始到行尾的部分，不被解释执行
  • 功能性语句
    • 操作系统命令
    • shell内部命令
    • 自编程序
    • 其它shell程序名等
    • 几个常用的功能性语句(命令):
      • read命令
        • read从标准输入读入一行, 并赋值给后面的变量
        • 其语法为：
          • read var
            • 把读入的数据全部赋给var
          • read var1 var2 var3
            • 把读入行中的第一个词(word)赋给var1
            • 第二个词赋给var2, ……
            • 把其余所有的词赋给最后一个变量
          • 如果执行read语句时标准输入无数据, 则程序在此停留等侯, 直到数据的到来或被中止运行
      • expr 命令
        • 算术运算命令expr主要用于进行简单的整数运算，包括加(+)、减（-）、乘（*)、整除（/）和求模（%）等操作
        • 例如:
          • $ expr 12 + 5 * 3
            　　27
            • 反斜线去掉*号的元字符含义
          • $ num=9
            $ sum=`expr $num * 6 `
            $ echo $sum
            54
          • 反撇号引用命令的运行结果
      • tput 命令
        • tput命令主要用于设置终端工作模式, 或读出终端控制字符.
        • tput命令与终端控制代码数据库terminfo相连, 根据shell环境变量TERM的值, 读出这种终端的指定功能控制代码
  • 结构性语句
    • 条件测试语句
    • 多路分支语句
    • 循环语句
    • 循环控制语句等

三、 结构性语句

  • 条件语句　if…then…fi
    • 语法结构
      • if　　表达式
      • then　命令表
      • fi
  • 条件语句　if…then…else…fi
    • 语法结构为
      • if　　表达式
      • then　命令表1
      • else　命令表2
      • fi
  • 测试语句 test
    • test语句可测试三种对象：
      • 字符串
        • test “$answer” = “yes”
      • 整数
        • test $num –eq 18
        • 变量num的值是否为整数18
      • 文件属性
        • test -d tmp
        • 测试tmp是否为一个目录名
    • 用法
      • 用法一：
        • test $name -eq $1
          echo $?
      • 用法二：
        • if test -f $filename
          then
          ……
          fi
        • 通常简写为: if [ -f $filename ]
  • 多路分支语句 case…esac
    • 多路分支语句case用于多重条件测试，语法结构清晰自然.其语法结构为:
      • case　　字符串变量　　in
      • 　　　　模式1）
      • 　　　　　　　　　　命令表1
      • 　　　　　　　　　　 ;;
      • 　　　　模式2）
      • 　　　　　　　　　　命令表2
      • 　　　　　　　　　　 ;;
      • 　　　　……
      • 　　　　模式n)
      • 　　　　　　　　　　命令表n
      • 　　　　　　　　　　 ;;
      • 　　　　esac
      • case语句只能检测字符串变量
      • 各模式中可用文件名元字符,以右括号结束
      • 命令表以单独的双分号行结束,退出case语句
      • 模式 n常写为字符* 表示所有其它模式
      • 最后一个双分号行可以省略
  • 循环语句 for…do…done
    • 当循环次数已知或确定时，使用for循环语句来多次执行一条或一组命令.　循环体由语句括号 do 和 done 来限定
    • for　　变量名　　in　　单词表
    • do
    • 　　　命令表
    • done

  • 循环语句　while…do…done

    • 语法结构为:
    • while　　命令或表达
    • do
    • 　　　　　命令表
    • done
    • while语句的退出状态为命令表中被执行的最后一条命令的退出状态

  • 循环语句　until…do…done

    • 语法结构为:
    • until　　命令或表达式
    • do
    • 　　　　命令表
    • done
  • 循环控制语句 break 和 continue
    • break语句
      • break 从包含该语句的最近一层循环中跳出一层
      • break n 则跳出n层;
    • continue语句
      • continue 马上转到最近一层循环语句的下一轮循环上
      • continue n则转到最近n层循环语句的下一轮循环上.
